# 🔐 Password Generator (Python)

A simple and customizable password generator built using Python.

This project generates secure passwords based on user-defined preferences such as length and character types.

---

## 📌 Features

- User-defined password length
- Option to include:
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Numbers (0–9)
  - Special characters (!@#$%^&* etc.)
- Random and secure password generation
- Console-based interaction

---

## 🧠 Concepts Used

- Python `random` module
- String manipulation
- Conditional statements
- Loops
- User input handling
- Logical flow control

---

## ▶️ How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Run the file using:

```bash
python password_generator.py
```
